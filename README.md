[Image](https://trello.com/b/k1vqoXPB/plm-web) uploaded to [Trello](https://trello.com/b/k1vqoXPB/plm-web)

## Installation ##

Requirements include:

- npm
- nodejs
- ruby gems
- sass

```
npm install -g bower
npm install -g gulp

gulp install
gulp
```

On your browser, you may access by ```http://localhost:8080```
